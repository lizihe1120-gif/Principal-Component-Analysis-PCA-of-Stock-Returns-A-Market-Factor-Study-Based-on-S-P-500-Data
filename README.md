# 股票收益率主成分分析（PCA）——基于S&P 500数据的市场因子研究
本项目使用R语言对S&amp;P 500成分股的日收益率数据进行主成分分析（PCA），旨在揭示市场共同因子对股票收益的驱动作用。项目涵盖了数据清洗、面板数据构建、收益率计算、PCA分析以及标准化前后的对比，并探讨了实体对齐问题在金融数据分析中的重要性。

## 数据说明

- **数据来源**：S&P 500成分股的历史日交易数据，包含以下字段：
  - `Date`: 交易日期
  - `Open`, `High`, `Low`, `Close`: 开高低收价格
  - `Volume`: 成交量
  - `Name`: 股票代码（如AAPL, GOOGL等）
- **时间范围**：原始数据覆盖较长时间，但分析仅使用2019年11月1日至2022年10月31日三年窗口。
- **数据预处理**：
  1. 剔除在分析窗口期内未完整交易的股票（首日>2019-11-01或末日<2022-10-31）。
  2. 找出所有股票共同存在的交易日（共755天）。
  3. 构建宽表：每行一个日期，每列一支股票，值为当日收盘价。
  4. 计算每日简单收益率：`(Close_t - Close_{t-1}) / Close_{t-1}`，删除第一行缺失值。

分析步骤
1. 数据加载与清洗
从stock_data.zip读取数据。

筛选出2019-11-01至2022-10-31期间数据完整的490支股票（共500支，10支因数据不足被剔除）。

确定755个共同交易日。

2. 构建收益率面板
将收盘价转换为宽格式，得到755×490的矩阵。

计算日收益率，得到754×490的收益率矩阵。

3. PCA分析（原始收益率）
对收益率矩阵（不含日期列）进行主成分分析。

提取前5个主成分的载荷矩阵。

计算各主成分的解释方差比例，绘制前20个主成分的碎石图，并用find_elbow函数识别拐点（图中标记为第2个主成分）。

绘制累计方差解释率曲线，标记累计解释率达到95%的主成分数量（第195个主成分）。

4. 标准化后的PCA分析
对收益率矩阵进行标准化（scale()），使每个股票收益率均值为0、方差为1。

重复PCA分析，比较标准化前后的结果。

标准化后，第一个主成分解释方差比例略有下降（从43.65%降至42.42%），碎石图拐点仍为第2个主成分，累计95%所需主成分数增加至216个。

标准化消除了不同股票原始波动幅度的影响，使PCA更关注相关性结构，适合识别系统性因子。

5. 实体对齐问题讨论
在研究报告部分（Q11）中，讨论了金融数据中常见的企业名称/代码不一致问题（实体对齐）。引用了Zeng等人(2021)提出的基于强化学习的集体实体对齐方法，阐述了错误对齐对PCA分析（如协方差矩阵失真）的影响，并指出高质量对齐能提高市场因子识别的可靠性。

主要结果

标准化PCA：

标准化后各主成分的方差更加分散，说明原始数据中高波动股票的贡献被平衡。
